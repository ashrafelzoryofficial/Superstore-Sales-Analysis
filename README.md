# Superstore Sales Analysis

## Project Overview
In this project, I analyzed the Superstore sales dataset to understand business performance and identify key insights.

## Objectives
The goal of this analysis is to explore sales data and answer important business questions such as:
- Which markets generate the most revenue?
- Which customer segments contribute the most sales?
- How do discounts affect profit?

## Steps in the Analysis
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Extracting Insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

## Key Insights
- The EU market generated the highest sales and profit.
- The Consumer segment contributes the most revenue.
- Office Supplies is the top category in sales.
- Large discounts can reduce profit.
- Standard Class is the most commonly used shipping mode.
